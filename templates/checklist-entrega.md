# Checklist de Entrega de Projeto

Use este checklist antes de abrir o Pull Request.

## Estrutura e arquivos

- [ ] Pasta criada no semestre correto (`projetos/AAAA-S/`)
- [ ] `README.md` do projeto preenchido
- [ ] Pasta `modelos/` com arquivos necessarios
- [ ] Pasta `slicer/` com perfis/arquivos de fatiamento relevantes
- [ ] Pasta `imagens/` com evidencias claras
- [ ] Pasta `referencias/` organizada
- [ ] Arquivo `LICENSE` na pasta do projeto

## Metadados obrigatorios

- [ ] Versao do modelo informada
- [ ] Escala informada
- [ ] Unidade de medida informada
- [ ] Licenca informada

## Qualidade tecnica

- [ ] Parametros de impressao registrados
- [ ] Tempo estimado de impressao registrado
- [ ] Consumo estimado de filamento registrado
- [ ] Instrucoes de montagem descritas

## Qualidade pedagogica

- [ ] Objetivo educacional claro
- [ ] Publico-alvo definido
- [ ] Justificativa pedagogica consistente
- [ ] Aplicacao em sala de aula descrita
- [ ] Limitacoes e melhorias futuras registradas

## Gestao de tamanho de arquivos

- [ ] Nao ha arquivos excessivamente grandes sem justificativa
- [ ] Arquivos intermediarios desnecessarios foram removidos
- [ ] Quando necessario, links externos foram documentados no README
