# Guia de Impressao 3D para a Disciplina

Guia introdutorio para apoiar decisoes tecnicas durante o desenvolvimento de recursos educacionais impressos em 3D.

## 1. Planejamento do modelo

- defina o conceito pedagogico antes da geometria final;
- projete pensando em montagem, manuseio e durabilidade;
- considere seguranca para o publico-alvo (arestas, partes pequenas, resistencia).

## 2. Parametros basicos de impressao (referencia)

Valores de partida para FDM (ajustar conforme material/equipamento):

- altura de camada: 0,16 mm a 0,24 mm;
- espessura de parede: 1,2 mm a 1,6 mm;
- infill: 10% a 30% conforme esforco mecanico;
- velocidade: 40 mm/s a 70 mm/s;
- bico: 0,4 mm (padrao).

## 3. Materiais

- PLA: recomendado para prototipos didaticos pela facilidade de impressao;
- PETG: indicado quando ha necessidade de maior resistencia termica/mecanica;
- outros materiais devem ser justificados no README do projeto.

## 4. Orientacao e suportes

- minimize suportes com ajustes de orientacao do modelo;
- prefira divisao em partes quando a geometria comprometer qualidade;
- documente no projeto a orientacao utilizada para replicabilidade.

## 5. Registro tecnico obrigatorio

Para cada versao publicada, registre:

- versao do modelo;
- escala;
- unidade de medida;
- licenca;
- slicer e versao do slicer utilizados;
- parametros principais aplicados.

## 6. Estimativas e validacao

Sempre inclua:

- tempo estimado de impressao;
- consumo estimado de filamento;
- resultados de pelo menos um teste fisico;
- eventuais ajustes para a proxima iteracao.
